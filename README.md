# 전자출입명부

## 프로젝트 소개
**전자출입명부**는 교육사업 운영 시 필요한 현장 관리 업무를 효율적으로 처리하기 위해 제작한 서비스입니다.  
출석 관리를 위한 **전자출입명부 기능**과, 교육 종료 후 참여자의 의견을 수집할 수 있는 **설문조사 기능**을 제공합니다.

실제 사업 운영에 활용하기 위해 개발되었으며, 현재 배포까지 완료된 상태입니다.

---

## 주요 기능

### 1. 전자출입명부
교육 참여자의 출석 여부를 등록하고 관리할 수 있는 기능입니다.  
현장에서 간편하게 출석 체크를 진행할 수 있도록 구성했습니다.

### 2. 교육 종료 후 설문조사
교육이 끝난 뒤 참여자를 대상으로 설문조사를 진행할 수 있는 기능입니다.  
교육 만족도 및 의견 수집에 활용할 수 있습니다.

---

## 화면 구성
메인 **Home 화면**에서 각 기능 버튼을 클릭하면 해당 화면으로 이동할 수 있도록 구현했습니다.  

또한, 우측 상단의 **햄버거 메뉴**를 통해서도 동일하게 화면 전환이 가능하도록 구성해  
사용자가 보다 편리하게 원하는 기능에 접근할 수 있도록 했습니다.

---

## 배포
서비스는 현재 **배포 완료** 상태입니다.  

- **배포 환경**: Azure
- **데이터베이스**: Azure Database 사용

사업 운영 중인 서비스이기 때문에, 배포 링크는 공개하지 않습니다.

---

## 프로젝트 목적
교육사업 현장에서 필요한 출석 관리와 사후 설문조사를 하나의 서비스로 통합하여  
운영 효율성을 높이고, 참여자 관리 및 피드백 수집 과정을 간소화하는 것을 목표로 했습니다.
